# GTA6 攻略网站

一个纯前端的 GTA6 中文攻略站示例，包含：

- 新手开荒清单（支持本地存储）
- 任务攻略检索（按类型筛选 + 关键词搜索）
- 任务详情弹窗（展示奖励与打法要点）
- 地图与势力说明
- 武器与载具推荐路线
- 刷钱流程与 FAQ

## 本地运行

本项目无需安装依赖，直接使用任意静态服务器即可：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`
