## 使用方式

### 方式1: 直接打开
```bash
# 在文件管理器中双击 index.html
```

### 方式2: 使用启动脚本
```bash
cd /opt/data/deliverables/gta6-guide
chmod +x start.sh
./start.sh
```

### 方式3: 手动启动服务器
```bash
cd /opt/data/deliverables/gta6-guide
python3 -m http.server 8000
# 或
npx http-server
```

然后在浏览器中访问: `http://localhost:8000`

## 自定义修改

### 修改颜色主题
编辑 `styles.css` 文件中的CSS变量：
```css
:root {
    --primary: #ff6b00;      /* 主要颜色 */
    --secondary: #00d4ff;    /* 次要颜色 */
    --accent: #ff00ff;       /* 强调颜色 */
    --dark: #0a0a0a;         /* 深色背景 */
    --light: #ffffff;        /* 浅色文字 */
}
```
